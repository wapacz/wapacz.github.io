26 September 2017 · Docker

Push own docker image

> mkdir ghost 
> nano ghost/Dockerfile

Put into the Dockerfile:

FROM ghost:1.8

> docker build ghost/
> docker tag ghost wapacz/ghost
> docker login
...
> docker tag ghost wapacz/ghost
> docker push wapacz/ghost
