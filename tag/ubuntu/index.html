Switching between JDK8 and JDK9 on Ubuntu

To switch to Java 8 run: sudo update-java-alternatives --set /usr/lib/jvm/java-1.8.0-openjdk-amd64 To switch to Java 9 run: sudo update-java-alternatives --set /usr/lib/jvm/java-1.9.0-openjdk-amd64 »
