NVM (Node.js) on Ubuntu

Installation of NVM

sudo apt-get update
sudo apt-get install build-essential libssl-dev
curl -sL https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh -o install_nvm.sh
bash install_nvm.sh
source ~/.profile

Listing remote node.js versions:

nvm ls-remote

If you have multiple Node.js versions, you can see what is installed by typing:

nvm ls

To install specific node.js version type:

nvm install 8.9.4

Usually, nvm will switch to use the most recently installed version. You can explicitly tell nvm to use specific version:

nvm use 8.9.4

Source: https://www.digitalocean.com/community/tutorials/how-to-install-node-js-on-ubuntu-16-04
